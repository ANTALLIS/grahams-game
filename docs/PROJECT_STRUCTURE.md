# Project Folder Structure Documentation

## Assets
Contains all static resources used in the game:
- **Fonts** – Custom and system fonts.
- **Images** – PNGs, JPEGs, textures.
- **Sprites** – Sprite sheets and individual animation assets.

## Docs
All project-related documentation:
- API references
- Diagrams and flowcharts
- Design notes and specifications

## Entities
Defines core game objects and interactive elements:
- **Player** – Player-controlled characters.
- **NPCs** – Non-player characters.
- **Environment objects** – Trees, buildings, and other interactive/decorative elements.

## Scenes
Holds game world setups and levels:
- Individual scene files or prefabs
- Layouts for game levels or areas

## Systems
Core game logic and management scripts:
- Input handling
- Game mechanics
- Rendering and physics systems
- Utility scripts supporting multiple entities or scenes
